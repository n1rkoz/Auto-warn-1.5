# Auto-warn-1.5

🛡️ Auto-warn-1.5
Мод для автоматической выдачи варнов за никнеймы по пункту правил 1.5.

📥 Установка:

🕹️ Установите Fabric 1.20.1.

📂 Переместите мод в папку mods.

🛠️ Скачайте Cloth Config и также закиньте его в папку mods.

📝 Скачайте конфиг с триггер-словами и переместите его по пути:

.\minecraft\labymod-neo\fabric\1.20.1\config

или .\minecraft\labymod-neo\modpacks\ваша_сборка\fabric\1.20.1\config 📂

🎮 Как использовать:

⚙️ Настройка: В настройках игры («Назначение клавиш») установите удобные кнопки для функций мода.

🕵️ Поиск нарушителей: Зайдите на сервер (анархию) и откройте GUI мода с помощью вашего бинда.

📋 Копирование: Если мод найдет запрещенный ник, он появится в списке. Нажмите кнопку «Скопировать», чтобы ник попал в буфер обмена. ✨

📜 Проверка истории: 

Нажмите бинд «Отправить history». Мод проверит историю последнего скопированного игрока на наличие активных варнов от других модераторов. Результат придет в чат! 💬

🔨 Выдача варна: Нажмите бинд «Варнуть»:

Если варна нет — мод автоматически выдаст наказание. ✅

Если варн уже висит — мод отменит действие и предупредит об этом в чате. ❌

Удачной модерации! ⚖️✨
