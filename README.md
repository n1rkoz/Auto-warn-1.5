# Auto-warn-1.5
Мод для автоматической выдачи варнов за никнеймы по пункту правил 1.5.

Установка:
Установите Fabric 1.20.1.

Переместите мод в папку mods.

Скачайте Cloth Config и также переместите в папку mods.

Скачайте конфиг с триггер-словами и переместите его по пути:

.minecraft\labymod-neo\fabric\1.20.1\config

или .minecraft\labymod-neo\modpacks\ваша_сборка\fabric\1.20.1\config

Как использовать:
Настройка: В настройках игры («Назначение клавиш») установите удобные клавиши для функций мода.

Поиск нарушителей: Зайдите на сервер анархию и откройте GUI мода с помощью назначенного бинда.

Копирование: Если мод обнаружит запрещенный ник, он появится в списке. Нажмите кнопку «Скопировать», чтобы сохранить ник в буфер обмена.

Проверка истории: При нажатии бинда «Отправить history» мод проверит историю последнего скопированного игрока на наличие активного варна от других модераторов. Результат (есть варн или нет) отобразится в чате.

Выдача варна: При нажатии бинда «Варнуть» мод проверит историю последнего скопированного игрока.

Если активного варна нет — мод автоматически выдаст наказание.

Если варн уже есть — мод отменит действие и сообщит об этом в чате
